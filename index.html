Ordered pair:


A pair of elements in a fixed order is called an ordered pair. An ordered pair pair having 'a' as the first element and 'b' as the second element is denoted as (a,b). The first element 'a' is called antecendent & the second element 'b' is called consequent.

Note:

{2,3} is a set, not an ordered pair

Equality of ordered pairs


Two ordered pairs (a,b) & (x,y) are equal if their corresponding elements match, ie, (a,b)=(x,y) if a=x and b=y.
Q. Find the values of x and y
  1. (x+y,y+2)=(7,4)
    Sol,
    Here,(x+4,y+2)=(7,4)
    By equality of ordered pairs,
    x+4=7 & y+2=4
    or,x=7-4 or,y=4-2
    ∴x=3 ∴y=2

    Hence, x=3 & y=2
